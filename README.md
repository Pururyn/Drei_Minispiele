# 🎮 Drei Minispiele

A collection of **three mini-games** built in **Unreal Engine (Blueprints)**, each exploring a different type of physics.

---

## 🕹️ Mini-games

### 🦾 Claw Machine — Constraints & Grab
Control a claw to grab the capycoco plushie. Watch out — grabbing an orange means you lose!

### 💥 Destruct Boxes — Impulse & Forces
Shoot and blow up boxes with explosive particles. Features a surprise.

### 🔮 Marble Maze — Collision & Gravity
Tilt a board to guide a marble through a maze and reach the goal — without falling off.
